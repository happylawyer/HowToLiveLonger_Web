# 《程序员延寿指南》公益版

## 1. 项目简介
本项目基于 GitHub 开源仓库 `geekan/HowToLiveLonger`，旨在通过 AI + 文档的形式，让复杂的医学证据变得通俗易懂，帮助更多人通过科学手段延长寿命。

## 2. 核心架构（白嫖版）
- **前端框架**：VitePress (高效、免费、美观)
- **部署平台**：Vercel / GitHub Pages
- **AI 脑子**：Google Gemini 1.5 Flash (API Key 已就绪)
- **交互方式**：文档查阅 + 右下角 AI 问答球
