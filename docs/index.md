# 程序员延寿指南 (公益网页版)

本项目是基于 GitHub [geekan/HowToLiveLonger](https://github.com/geekan/HowToLiveLonger) 的开源内容制作的公益展示页面。

---

## 1. 术语
* ACM: All-Cause Mortality / 全因死亡率

## 2. 目标
* 稳健的活得更久
* 减少工作时间

## 3. 关键结果
* 降低 66.67% 全因死亡率
* 增加 ~20 年预期寿命

## 4. 核心行动建议

### 输入 (Input)
* **固体**：吃白肉、蔬果为主，多吃辣，多吃坚果。
* **液体**：喝咖啡、牛奶、茶；**戒酒**，少喝甜味饮料。
* **气体**：不吸烟。
* **光照**：多晒太阳。

### 输出 (Output)
* **运动**：每周 3 次挥拍运动（羽毛球/乒乓球）。
* **睡眠**：每天睡 7 小时，推荐 22-24 点入睡。
