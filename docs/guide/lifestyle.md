# 🏃‍♂️ 办公室补救：高效运动与习惯

白领久坐是健康的第一杀手，我们需要针对性的补救措施。

---

## 高效运动方案
*   **挥拍类运动**（羽毛球、乒乓球）：性价比之王，降低 **47%** 全因死亡率。
*   **剧烈运动 (HIIT)**：每周仅需 15-20 分钟，降低 **16-40%** 死亡风险。

## 职场微习惯
*   **光照**：午休晒太阳。多晒太阳 ACM 降低约 **40%**。
*   **口腔卫生**：认真刷牙。过早死亡风险降低 **25%**。
*   **泡澡**：每天热水澡，心血管总风险降低 **28%**。

## 睡眠管理
*   **22-24 点入睡**：被认为是最佳的睡眠窗口。
*   **规律性**：保持每天 7 小时的稳定时长，不要报复性补觉。
