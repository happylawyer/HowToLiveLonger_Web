# 延寿指南核心内容

本项目是基于 GitHub [geekan/HowToLiveLonger](https://github.com/geekan/HowToLiveLonger) 的开源内容制作的公益展示页面。

---

### 1. 术语
* **ACM**: All-Cause Mortality / 全因死亡率

### 2. 目标
* 稳健的活得更久
* 减少工作时间

### 3. 关键结果
* 降低 **66.67%** 全因死亡率
* 增加 **~20 年** 预期寿命

---

### 4. 核心行动建议 (Action)

#### 🥗 输入 (Input)
* **固体**：吃白肉（-11%~-3% ACM）、蔬果为主（-26%~-17% ACM），多吃辣（-23% ACM），多吃坚果（-27%~-4% ACM）。
* **液体**：喝咖啡（-22%~-12% ACM），喝牛奶（-17%~-10% ACM），喝茶（-15%~-8% ACM）。**戒烟戒酒**。
* **光照**：晒太阳（-~40% ACM）。

#### 🎾 输出 (Output)
* **运动**：每周 3 次 45 分钟挥拍运动（-47% ACM）。
* **日常**：刷牙（-25% ACM）、泡澡（-28% 心血管风险）。
* **睡眠**：每天睡 7 小时全因死亡率最低。
